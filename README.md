# Frontend FarmaciaHcodelab

> Frontend da aplicacao de farmacia do curso HcodeLab Web Professional, desenvolvido com Next.js, React e integracao com API AdonisJS.

## Stack

- Next.js
- Axios

## Prerequisitos

- Node.js v14+

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/frontend-farmaciaHcodelab.git
cd frontend-farmaciaHcodelab

# Instale as dependencias
npm install

# Inicie a aplicacao
npm run dev
```
